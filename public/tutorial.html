04 / API 配置指南

如何获取 API Key?

1

进入控制台

登录本站，点击左侧导航栏的 API 图标进入开发者中心。

2

创建令牌 (Token)

在左侧菜单选择 「令牌管理」，点击新建按钮。
设置：名称随意填写，分组选择“默认分组”。

3

复制密钥

创建成功后，你将获得一串以 sk- 开头的密钥。这是调用服务的唯一凭证。

请妥善保管，密钥仅在创建时显示一次
